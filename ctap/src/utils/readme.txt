This folder contains external tools that are needed in some parts of the pipeline.

External tools that are actively developed at BWRC and used also by other projects than CTAP are not included.
They are loaded from the "ukko" server or a local copy of the respective repository.

Toolboxes:
LIBRA 		Robust data analysis methods (e.g. MCD)
mksqlite 	Access to SQLite databases (measurement config file)
continuous_sound_and_vibrations_analysis 	To enable MCD without statistics toolbox. Temporary addition.
